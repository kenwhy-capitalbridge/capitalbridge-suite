### Advisory journey (conceptual order)

1. **Forever Income** — sustainability of lifestyle spend vs capital stack.  
2. **Income Engineering** — income structure, unlocking capital, allocation.  
3. **Capital Health** — durability of withdrawals and capital over time.  
4. **Capital Stress** — resilience under stressed return paths.  
5. **Strategic execution** (where your plan includes it) — deeper execution tier on the platform.

### Membership & trials

- **Trial** and campaign passes may show a **limited “System Insight”** area instead of the full **Lion’s Verdict** in the model apps. Paid tiers unlock the full verdict and server save where entitled.  
- **PDF reports** may omit full Lion narrative on trial-class plans; export availability follows your current membership.

### Getting help

- Use **BACK** in the model header to return via the platform session sync when offered.  
- Account, membership, and checkout happen on the **login** app; the **platform** app is the main signed-in home for the framework.
