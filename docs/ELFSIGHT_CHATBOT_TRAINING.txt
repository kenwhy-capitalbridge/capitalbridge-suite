INTENT 1 — SIGN UP / FIRST ACCESS (after payment)
================================================================================
User may say: just paid, no email, first time, set password, how to login.

STEP 1 — Bot:
“After payment, check the email you used at checkout — including spam. You’ll get a link to set your password. Then use Account Access to sign in.”

STEP 2 — Ask:
“Have you seen any email from us in that inbox yet?”

IF YES →
“Use only the newest link. If the page says the link can’t be used, enter your email and tap Send Me A New Link, then check email again.”

IF NO →
“Wait a few minutes, check spam. If you’re on the payment-complete screen, use Send Password Setup Email if it’s there — it uses the email we have for that payment.”

IF WRONG INBOX →
“We send to the email from checkout. You need that inbox, or contact support with payment details.”

Pages: /checkout → pay → payment return; then /access after password set.

================================================================================
INTENT 2 — LOGIN ISSUES
================================================================================
STEP 1 — Bot:
“On Account Access, use the same email as checkout and your password. If you never set a password, tap Send Me A New Link first.”

STEP 2 — Ask:
“Have you already set your password using the email link?”

IF NOT YET →
“Tap Send Me A New Link, open the newest email, set your password, then sign in.”

IF YES →
“Check caps lock. If it still fails, use Forgot password or Send Me A New Link.”

IF TOO MANY TRIES →
“Wait a minute, then try once more — the page slows down after many attempts.”

Primary button on page: Access Platform. Secondary: Send Me A New Link.

================================================================================
INTENT 3 — PASSWORD RESET / LINK PROBLEMS
================================================================================
STEP 1 — Bot:
“Use Send Me A New Link on Account Access, or the Forgot password page. There’s a short wait before you can send again.”

STEP 2 — Ask:
“Do you see a message that the link can’t be used?”

IF YES →
“Enter your email on that page, tap Send Me A New Link, open only the newest email.”

IF NO (normal sign-in) →
“Use Send Me A New Link under the form, or Forgot password. Check spam.”

IF MULTIPLE TABS →
“Close old tabs. Only use the latest email link.”

Cooldown: about 45 seconds between sends (user sees a countdown).
