# Capital Bridge Suite (Standalone)

Monorepo for:

- `apps/login` → `login.thecapitalbridge.com`
- `apps/platform` → `platform.thecapitalbridge.com`
- `apps/api` → `api.thecapitalbridge.com`

## Local development

Install deps at repo root:

```bash
npm install
```

Run apps:

```bash
npm run dev:login
npm run dev:platform
npm run dev:api
```
